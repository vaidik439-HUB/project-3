# Collector Manipulator

A simple Python console-based application to manage student records using dictionaries.  
This program allows users to add, display, update, and delete student information interactively.

---

## Features

- Add new student records
- Display all students
- Update student information
- Delete student records
- Display available subjects
- Menu-driven interface

---

## Technologies Used

- Python 3
- Dictionary Data Structure
- Loops and Conditional Statements

---

## Program Structure

Each student record contains:

- Student ID
- Name
- Age
- Grade
- Date of Birth
- Subjects

Data is stored in a Python dictionary using the Student ID as the key.

---

## Menu Options

| Option | Description |
|--------|-------------|
| 1 | Add Student |
| 2 | Display All Students |
| 3 | Update Student Information |
| 4 | Delete Student |
| 5 | Display Subjects Offered |
| 6 | Exit Program |

---

## Code Overview

### Add Student
Stores student information in the dictionary.

### Display Students
Displays all saved student records.

### Update Student
Updates the grade of an existing student.

### Delete Student
Removes a student record using Student ID.

### Subjects Offered
Displays predefined subjects.

### Exit 
Exiting  the program with the exiting massage.
